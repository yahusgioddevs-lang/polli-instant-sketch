# Instant Polli Sketch

Turn words into images in one click. Type a prompt, pick a size, hit Generate — powered by Pollinations `flux`. No sign-up, no clutter, just fast image generation in a minimal web UI. Built as a lightweight reference implementation showing how to call the Pollinations Image API from a simple Python backend.

## Run

```bash
export POLLINATIONS_API_KEY=sk_...
python3 app.py
```

Open http://localhost:8080
